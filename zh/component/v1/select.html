# Select 属性

参数 说明 类型 是否必填 可选值 默认值
viewObj 当前select的jquery对象 object 是 - -
multiple 是否允许多选 boolean 否 true/false false
allowClear 是否允许清空选项 boolean 否 true/false false
isRemote 是否远程加载 boolean 否 true/false false
url url远程加载地址，必须与isRemote同时使用 string 否 - -
placeholder 描述输入字段预期值的提示信息 string 否 - -
language 配置提示信息详见language配置 object 否 - -

# Select 方法

方法名称 说明 参数 返回值
data 获取当前所有的选项 - object[]

# Select 事件

事件名称 说明 参数
change 选中值发生变化时触发 select对象

# Language

# 属性

参数 说明 类型 是否必填 可选值 默认值
noResults 无数据提示 fun 否 - 未找到结果
errorLoading 加载错误提示 fun 否 - 无法载入结果

# 友情提示

警告

  1. noResults 是程序正常没有数据的提示
  2. errorLoading 是程序异常加载出错的提示
