# Grid 表格

用于展示二维的数据，可对数据进行排序、筛选、对比或其他自定义操作。

# Grid 属性

参数 说明 类型 是否必填 可选值 默认值
gridObj 当前table的jquery对象 object 是 - -
colNames 列名称 string 是 - -
colModel 列模型,详见ColModel object[] 是 - -
multiselect 是否允许多选 boolean 否 true/false true
caption 表格标题，underfine时不显示标题 string 否 - -
rowNum 显示数据总条目数（当前分页） number 否 - 10
showPager 是否显示分页栏 boolean 否 true/false true
sortorder 表格列排序的方向（升/降） string 否 desc/asc desc
sortname 默认表格列排序的列名 string 否 - -
dynamicConditionQuery 动态条件查询，DynamicConditionQuery object 否 - -

# Grid 方法

方法名称 说明 参数 返回值
reloadGrid 重新加载当前表格，也会向服务器发起新的请求 object(ReloadGrid 参数) -
dynamicConditionLoad 动态条件查询的窗口配置 object(DynamicConditionLoad 参数) -
exportExcel 导出Excel object(ExportExcel 参数) -

# Grid 事件

事件名称 说明 参数
change 选中值发生变化时触发 select对象

# ColModel

# ColModel 事件

事件名称 说明 参数 返回值
editableExt 该列会根据函数的返回值是否为true来确认是否编辑，必须与ediable配合使用 function（输入参数是rowObj） true/false

# ColModel 参数

参数 说明 类型 是否必填 可选值 默认值
align 定义数据相对单元格的对齐方式 string 否 left/center/right left
displayType 非静态下显示的元素类型 string 否 date/select/checkbox -
editable 是否可编辑 boolean 否 true/false false

# DynamicConditionQuery 参数

参数 说明 类型 是否必填 可选值 默认值
active 是否启用 boolean 否 true/false -
multipleSearch 是否允许多列查询 boolean 否 true/false true
multipleGroup 是否允许组合查询(与或非组合使用) boolean 否 true/false false

# DynamicConditionLoad 参数

参数 说明 类型 是否必填 可选值 默认值
url 远程请求的url string 是 - -
postData 远程请求 object 否 - -

# ExportExcel 参数

参数 说明 类型 是否必填 可选值 默认值
url 远程请求的url string 是 - -
all 是否所有数据 boolean 否 true/false false
count 导出数据条数(只有当参数all为true时生效) int 否 - -

# ReloadGrid 参数

参数 说明 类型 是否必填 可选值 默认值
url 远程请求的url，如果为空时，不进行远程加载 string 否 - -
datatype 远程求数据类型与url配合使用 string 否 json/xml json
page 重加载后显示第几页 int 否 >0 1
