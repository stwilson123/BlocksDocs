# 属性

参数 说明 类型 可选值 默认值
v-model 绑定值 boolean / string / number — —
multiple 是否多选 boolean — false
disabled 是否禁用 boolean — false
size 输入框尺寸 string medium/small/mini —
clearable 是否可以清空选项 boolean — false
collapse-tags 多选时是否将选中值按文字的形式展示 boolean — false
multiple-limit 多选时用户最多可以选择的项目数，为 0 则不限制 number — 0
name select input 的 name 属性 string — —
autocomplete select input 的 autocomplete 属性 string — off
placeholder 占位符 string — 请选择
filterable 是否可搜索 boolean — false
allow-create 是否允许用户创建新条目，需配合 filterable 使用 boolean — false
filter-method 自定义搜索方法 function — —
remote 是否为远程搜索 boolean — false
remote-method 远程搜索方法 function — —
remote-url 远程请求url string — —
remote-params 远程请求参数 object — —
loading 是否正在从远程获取数据 boolean — false
loading-text 远程加载时显示的文字 string — 加载中
no-match-text 搜索条件无匹配时显示的文字，也可以使用slot="empty"设置 string — 无匹配数据
no-data-text 选项为空时显示的文字，也可以使用slot="empty"设置 string — 无数据
popper-class Select 下拉框的类名 string — —
reserve-keyword 多选且可搜索时，是否在选中一个选项后保留当前的搜索关键词 boolean — false
popper-append-to-body 是否将弹出框插入至 body 元素。在弹出框的定位出现问题时，可将该属性设置为 false boolean — true
options-data 绑定数据(查看格式) array — —
item-value 与options-data id 保持一致 string — id
item-text 与options-data text 保持一致 string — text

# 事件

事件名称 说明 回调参数
blocks-change 选中值发生变化时触发 目前的选中值 grid 编辑有3个回调参数 val,id,comp
blocks-visible 下拉框出现/隐藏时触发 出现则为 true，隐藏则为 false grid 编辑有3个回调参数 val,id,comp
blocks-remove 多选模式下移除tag时触发 移除的tag值 grid 编辑有3个回调参数 event,id,comp
blocks-clear 可清空的单选模式下用户点击清空按钮时触发 grid 编辑有3个回调参数 event,id,comp
blocks-blur 当 input 失去焦点时触发 grid 编辑有3个回调参数 event,id,comp
blocks-focus 当 input 获得焦点时触发 grid 编辑有3个回调参数 event,id,comp
blocks-init grid 编辑初始化触发 id,comp

方法名 说明 参数
focus 使 input 获取焦点 —
