# Grid 表格

# 示例

代码

<div>
    <bl-grid ref="AgGrid" :grid-options="gridOptions" unique-key="ID" @gridData="gridData"></bl-grid>
</div>

# Grid 属性

参数 说明 类型 可选值 默认值
grid-options 表格属性 (查看属性) object — —
grid-type 表格类型 string server/client server
no-load-data 初始化不加载数据 boolean true / false false
unique-key 表格rows数据唯一键 string — Id
grid-class 表格样式类 string — —
grid-style 表格样式 string — —
page-class 分页样式类 string — —
page-style 分页样式 string — —
columns-auto-fit 是否自动适应 boolean true / false true

# GridOptions

# grid-options属性

参数 说明 类型 可选值 默认值
columnDefs 列定义(查看属性) array — —
defaultColDef 默认列定义(查看属性) object — —
sortingOrder 排序设置 string[] asc / desc / null —
rowSelection 选择类型 string 'single'/'multiple' single
singleClickEdit 单击编辑 boolean true / false false
editType 编辑类型 string 'fullRow' —
showPagination 是否分页 boolean true / false true
paginationPageSize 分页每页条数 number — 10
floatingFilter 显示浮动筛选 boolean true / false true
