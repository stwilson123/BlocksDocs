# 移动端发布说明

# 介绍

移动端部署帮助文档

注意

运行环境 : Node.js , Android SDK, Java JDK , Gradle

node.js v10.15.3

  1. 下载：https://nodejs.org/zh-cn/download/
  2. 安装。
  3. 查看node.js版本号：
    1. win+R
    2. cmd
    3. 输入node -v 回车
  4. 查看npm 版本号：
    1. 输入npm -v 回车

Android SDK

  1. 下载: https://developer.android.google.cn/studio/

  2. 安装https://www.cnblogs.com/gufengchen/p/10991886.html

      例如SDK装在C:\Users\chenll\AppData\Local\Android\Sdk中，则环境变量为：

    3. 把%ANDROID_HOME%\platform-tools;%ANDROID_HOME%\tools​​​​​​​添加到Path环境变量中。

# 发布

  1. 打开VS Code
  2. 打开项目web文件夹(Blocks.Web)
  1. 打开移动端开发控制台

警告

控制台盘符路径如图：

  1. 输入npm install 安装需要的模块
  1. 输入npm run build:android
  1. 输入npx cap open android 打开Android项目
  1. 进入Android Studio 在控制栏中找到 File --> Sync Project with Gradle Files 安装Gradle依赖
  1. 在Android Studio 在控制栏中找到 Build --> Make Project 生成 .Apk 文件
  1. 在当前项目中找到.Apk文件并安装在手机上进行测试
