# PC端开发说明

# 介绍

基于vue.js框架开发

# 安装运行环境

node.js v10.15.3

  1. 下载：https://nodejs.org/zh-cn/download/
  2. 安装。
  3. 查看node.js版本号：
    1. win+R
    2. cmd
    3. 输入node -v 回车
  4. 查看npm 版本号：
    1. 输入npm -v 回车

警告

请注意：

  1. node 版本必须为 v10.15.3
  2. npm 版本必须为 6.4.1

# 安装开发工具

  1. 下载：https://code.visualstudio.com/
  2. 安装。

# 开发运行步骤

  1. 打开VS Code
  2. 打开项目web文件夹(Blocks.Web)
  1. 打开终端控制台

警告

控制台盘符路径如图：

  1. 输入npm install 安装需要的模块
  1. 输入npm run dev 运行开发环境
  1. 使用VS 2017打开项目
  2. 运行项目(端口号必须为8000)
  1. 浏览器输入：http://localhost:8080/Modules/Blocks.LayoutModule/dist/index.html#/ 进入开发环境

# 开始开发

  1. 使用VS 2017开发，并保持VS Code不关闭，增/删/改文件(js/css/html等)浏览器页面会自动刷新。
