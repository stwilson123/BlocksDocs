# 移动端开发说明

# 介绍

移动端开发帮助文档

注意

运行环境和开发工具与PC端开发一致，请参照 PC端开发说明

# 安装Git

  1. 下载：https://git-scm.com/
  2. 安装。

# 开发运行步骤

  1. 打开VS Code
  2. 打开项目web文件夹(Blocks.Web)
  1. 打开移动端开发控制台

警告

控制台盘符路径如图：

  1. 输入npm install 安装需要的模块
  1. 输入npm run serve 运行开发环境
  1. 浏览器输入：http://localhost:8081 进入开发环境

# 开始开发

  1. 在模块web项目中新建mobile->src->views文件夹
  1. 在mobile文件夹中新建 module.ts和routeProvider.ts文件
  1. 新建页面 每个页面一般包含三个文件。如：index.bl.html、index.bl.ts、index.bl.css(如果没有新增样式，此文件不需要创建)

注意

  1. html文件用来写html代码
  2. ts文件用来写ts代码
  3. css文件用来写css代码
  1. 在routeProvider.ts文件中配置路由
  1. 查看开发中的页面
