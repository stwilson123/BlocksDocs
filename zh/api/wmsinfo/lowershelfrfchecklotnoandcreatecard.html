# LowerShelfRFCheckLotNoAndCreateCard

正向下架批次信息验证并将验证通过的批次信息补全后返回生成卡片数据

# 接口注意事项

只有"执行"状态下的单据才可以进行采集操作 若此下架类型启用了策略控制，则会根据下架指导进行数据有效性的判断，以确认采集的数据是否合法 若无策略控制则根据明细中物料与批次信息进行有效性验证
