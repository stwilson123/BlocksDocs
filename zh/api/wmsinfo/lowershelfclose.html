# LowerShelfClose

下架单关单 修改下架单及明细状态为关闭状态

# 接口注意事项

只能关闭执行状态下的下架单 支持批量关单 关闭后释放下架指导中对批次库存的预占用

# 接口 参数

参数 类型 说明
CommonEntity CommonEntity 入参
RtnData RtnData 出参

# CommonEntity

参数 说明 类型 必填 可选值 默认值
ID 下架单Id，未使用 string 否 - -
IDs 下架单Id列表 List<string> 是 - -

# RtnData

参数 说明 类型 必填 可选值 默认值
RtnCode 返回值 string 是 1-成功;2-失败 -
RtnNote 返回说明 string 否 - -
